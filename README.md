# Advanced Deep Learning

## Course Overview
This repository contains my coursework, exercises, and projects from the
**Advanced Deep Learning** course
(2nd Semester, TU Dortmund University, Germany).

The course focuses on challenging, research-oriented deep learning problems
with a strong emphasis on practical skills and hands-on problem solving
using the PyTorch framework.

## Tools & Frameworks
- Python
- PyTorch
- Jupyter Notebooks

## Repository Structure
Each folder represents a separate exercise or project from the course.
